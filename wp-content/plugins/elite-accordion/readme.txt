== Installation ==

1. Install as a regular WordPress plugin.
2. You have to use shortcode for generate accordion in post or page. Example shortcode is given below.
[eliteaccordion][elitetoggle title="Title"]...[/elitetoggle][/eliteaccordion]
3. Replace the "Title" with your title and Replace the dot with your content. That's it. so easy.
